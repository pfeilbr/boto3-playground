# boto3-playground

learn [boto3](https://github.com/boto/boto3), the Amazon Web Services (AWS) SDK for Python

```sh
# install virtual env and dependencies
pipenv install

# activate python virtual env (optional)
pipenv shell

# run on change
make dev
```
